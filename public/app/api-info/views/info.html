API Documentation

Ter info: slechte API, neem hier zeker geen voorbeeld aan. Een goede API is self documented en kan je door navigeren met behulp van URLs.

GET

Lijst van verschillende auteurs 

    get /api/author
    
    response:
    [
        {
            _id: [string],
            meta: {
                likes: [number]
            }
        }
    ]

Detail auteur 

    get /api/author/:authorId
    
    vb. 
    /api/author/5421a81f0b4e83940fea61aa
    
    response:
    {
        _id: [string],
        meta: {
            likes: [number]
        }
    }

Recepten van een auteur 

    /api/author/:authorId/recipes
    
    vb.
    /api/author/5421a81f0b4e83940fea61aa/recipes
    
    response:
    {
        _id: [string],
        recipes: [
            {
                _id: [string],
                description: [string],
                title: [string],
                ingredients: [
                    {
                        name: [string],
                        amount: [number],
                        unit: [string]
                    }
                ]
            }
        ]
    }

Ingrediënten van een recept 

    get /api/author/:authorId/recipes/:recipeId/ingredients
    
    vb.
    /api/author/5421a81f0b4e83940fea61aa/recipes/5421aa55f7e37bac149a7bf7/ingredients
    
    response:
    [
        {
            name: [string],
            amount: [number],
            unit: [string]
        }
    ]

POST

Auteur toevoegen 

    post /api/author
    
    Request body:
    name=[string]

Recept toevoegen 

    post /api/author/:authorId/recipes
    
    request body:
    title=[string]&description=[string]
    
    optional pass ingredients
    title=[string]&description=[string]&ingredients=[arrayOfIngredients={name=[string]&amount=[number]&unit=[string]}]

Ingrediënt aan recept toevoegen 

    post /api/author/:authorId/recipes/:recipeId/ingredients
    
    vb.
    /api/author/5421a7c7ddaccaa40fb5fcd2/recipes/5421aa55f7e37bac149a7bf7/ingredients
    
    Request body:
    name=[string]&amount=[number]&unit=[string]
